너는 파이썬 Streamlit 라이브러리와 데이터 시각화(Plotly, Folium)에 정통한 시니어 프론트엔드/데이터 엔지니어이다. 
 즉시 실행 가능한 Streamlit 파이썬 코드(`app.py`)를 작성해 줘.

[현재 프로젝트 폴더 구조 및 환경]
1. 현재 내 프로젝트 루트 기준 `data/` 폴더 내에 실제 크롤링 및 분석된 데이터를 담은 `news.db`와 `economic_keyword_data.db` (SQLite)가 존재한다.
2. 작성해 줄 `app.py`는 프로젝트 루트 디렉토리에 위치한다고 가정하고, DB 연결 경로를 `data/news.db`, `data/economic_keyword_data.db`로 설정해라.

[기본 설정 및 테마]
1. 페이지 레이아웃은 'wide'로 설정하고, 전체적인 테마는 라이트 모드(Light Theme) 기반의 깔끔한 스타일을 유지해라.
2. Streamlit의 기본 컴포넌트로 구현하기 힘든 예쁜 카드 디자인이나 뱃지(긍정/부정) 등은 `st.markdown(..., unsafe_allow_html=True)`와 CSS를 적극 활용해서 구현해라.

[데이터 연동 구조 (가장 중요)]
1. 임의의 더미 데이터(np.random 등)를 차트 생성 코드에 직접 하드코딩하지 마라.
2. 대신 `get_metrics_data()`, `get_map_data()`, `get_chart_data()` 처럼 화면 구역별로 데이터를 불러오는 함수를 상단에 명확히 분리해서 만들어 줘. 
3. 각 데이터 로드 함수 내부에는 `sqlite3`와 `pandas`를 사용하여 `data/news.db` 또는 `data/economic_keyword_data.db`에 연결하는 기본 뼈대 코드와 임의의 SELECT 쿼리문을 작성해라. (내가 나중에 쿼리문만 수정해서 쓸 수 있도록)
4. 각 함수가 반환해야 하는 Pandas DataFrame의 형태(컬럼명, 데이터 타입)를 주석으로 상세히 명시해라.
5. UI와 차트를 그리는 코드는 오직 이 함수들이 반환한 DataFrame만을 바탕으로 동작해야 한다.

[화면 레이아웃 및 세부 요구사항]
화면은 크게 1) 좌측 사이드바, 2) 상단 메트릭, 3) 중앙 지도/리스트, 4) 중단 콤보 차트, 5) 하단 상세 분석 영역으로 나뉜다.

1. 사이드바 (Sidebar)
- 타이틀: "지능형 지역 경제 & 자산 분석"
- 필터: '분석 기간' (st.date_input)
- 자산 종류: 라디오 버튼 (코스피,코스닥)
- 지역 선택: '전국, 서울, 부산, 경기...' 등이 세로로 나열된 리스트 또는 라디오 버튼. 

2. 상단 메트릭 (Top Metrics)
- `st.columns(4)`를 사용하여 4개의 핵심 지표 카드를 배치.
- 카드 1: 종합 감성지수 / 카드 2: 변동성(Σ) / 카드 3: 코스피 변동/ 카드 4: 코스닥 변경

3. 중앙 구역 (Map & Top 10 List)
- 비율이 [1.5 : 1]인 두 개의 컬럼으로 분할.
- 왼쪽(지도): Folium 기반의 대한민국 지도를 렌더링하고(`streamlit-folium` 사용), DataFrame에 있는 위경도 데이터를 바탕으로 마커를 찍고 커스텀 툴팁(Popup)을 열어라.
- 오른쪽(리스트): DataFrame 기반으로 '지역별 핵심 이슈 TOP 10' 리스트를 출력. 긍정은 초록색 배경의 '▲ 긍정' 뱃지, 부정은 빨간색 배경의 '▼ 부정' 뱃지를 CSS로 적용.

4. 중단 구역 (Combo Chart)
- Plotly(`plotly.graph_objects`)를 사용하여 이중축 차트 생성. (X축: 날짜, Y1축: 지역 감성 지수 막대, Y2축: 자산 가격 선 그래프)

5. 하단 구역 (상세 분석 탭)
- `st.tabs`를 사용하여 [상관관계 분석, 감성 타임라인, 자산 가격 추이, 감성 기반 뉴스] 4개의 탭 생성.
- 상관관계 분석 탭 내부: 
  - 왼쪽: Plotly 기반 상관계수 히트맵 + 피어슨 상관계수 요약 데이터프레임(`st.dataframe`).
  - 오른쪽: Plotly 산점도 (추세선 포함) + 최신 감성 뉴스 리스트(커스텀 HTML 박스 적용).
